Pellet Reasoner Plug-in for Protégé 4.x

This plug-in makes OWL-DL reasoner Pellet available in Protégé 4.x. Pellet is a dual-licensed open source Java-based OWL-DL reasoner developed and commercially supported by Complexible Inc.

The version of this plug-in is bundled with Pellet 2.3.0.

Changes in the version 2.2.0 of the plugin are:

  • Upgraded the plugin to work with Pellet 2.3.0

Thanks for using Pellet.
